# Codestarters AI Bootcamp

The landing page + student portal for the **Codestarters AI Bootcamp** — a 7-day hands-on program where students go from "AI is magic" to shipping their own AI apps, agents, and automations. Sponsored by TrueFoundry.

## Files

- **`index.html`** — the entire site. Self-contained (no build step, no dependencies). Open it directly or host it anywhere static (GitHub Pages, Netlify, Vercel).

## Student portal

The site includes a login-gated student portal. The portal's content is **AES-GCM encrypted** and only decrypts client-side when the correct credentials are entered (key derived via PBKDF2 from `username:password`). The portal contains daily materials, the Discord link, and a working project-submission form.

> **Note:** This is client-side-only encryption — strong enough to gate cohort materials, but not for truly sensitive data. A backend login is the next step if needed.

## Project submissions

The submission form posts to a [FormSubmit](https://formsubmit.co) relay, which emails entries to the organizer — no backend required. The first submission triggers a one-time activation email that must be confirmed.

## Editing the portal

The encrypted portal blob in `index.html` is generated by `encrypt.mjs` (kept **out** of this repo because it holds the plaintext credentials). To change the portal contents or password, edit `encrypt.mjs` locally, run `node encrypt.mjs`, and paste the output into the `VAULT` object in `index.html`.
